# 人工智能与机器学习基础概念详解

## 一、人工智能概述

### 1.1 人工智能的定义
**人工智能（Artificial Intelligence, AI）** 是计算机科学的一个分支，旨在开发能够模拟、延伸和扩展人类智能的理论、方法、技术及应用系统。

**核心特征**：
- **感知能力**：通过传感器获取环境信息
- **推理能力**：基于规则或概率进行逻辑推理
- **学习能力**：从数据中提取模式，改进性能
- **适应能力**：根据环境变化调整行为
- **创造能力**：生成新的内容或解决方案

### 1.2 人工智能的发展历程
**三阶段发展**：
1. **符号主义AI（1950s-1980s）**：基于逻辑推理和知识表示
   - 代表：专家系统、逻辑编程
   - 局限：知识获取瓶颈，难以处理不确定性

2. **连接主义AI（1980s-2010s）**：基于神经网络和统计学习
   - 代表：感知机、反向传播算法
   - 突破：处理复杂模式识别问题

3. **深度学习时代（2012至今）**：大规模神经网络
   - 代表：卷积神经网络（CNN）、生成对抗网络（GAN）
   - 特征：端到端学习，自动特征提取

### 1.3 人工智能的分类
**按能力分类**：
- **弱人工智能（Narrow AI）**：专注于特定任务
  - 示例：图像识别、语音助手、推荐系统
  - 现状：已广泛应用，技术相对成熟

- **强人工智能（General AI）**：具备人类水平的通用智能
  - 特征：跨领域学习、自主推理、情感理解
  - 现状：仍在研究阶段，尚未实现

- **超人工智能（Superintelligence）**：超越人类所有认知能力
  - 探讨：技术奇点、伦理挑战
  - 现状：理论探讨为主

**按技术方法分类**：
- **符号AI**：基于规则和逻辑
- **统计AI**：基于概率和统计模型
- **连接主义AI**：基于神经网络
- **进化计算**：基于遗传算法
- **行为主义AI**：基于智能体与环境交互

## 二、机器学习基础

### 2.1 机器学习的定义
**机器学习（Machine Learning, ML）** 是人工智能的一个子领域，研究如何使计算机系统能够从数据中自动学习和改进，而无需显式编程。

**核心思想**：通过算法从数据中学习模式，并对新数据做出预测或决策。

### 2.2 机器学习的三要素
1. **模型（Model）**：学习的函数或假设空间
   - 线性模型：$y = w^Tx + b$
   - 非线性模型：神经网络、决策树
   - 概率模型：贝叶斯网络、隐马尔可夫模型

2. **损失函数（Loss Function）**：衡量预测与真实值的差异
   - 回归任务：均方误差（MSE）、平均绝对误差（MAE）
   - 分类任务：交叉熵损失、对数损失
   - 排序任务：对比损失、三元组损失

3. **优化算法（Optimization Algorithm）**：最小化损失函数的方法
   - 梯度下降法：随机梯度下降（SGD）、小批量梯度下降
   - 二阶方法：牛顿法、拟牛顿法
   - 自适应方法：Adam、RMSprop

### 2.3 机器学习的基本流程
**标准工作流**：
1. **问题定义**：明确任务类型（分类、回归、聚类等）
2. **数据收集**：获取相关数据集
3. **数据预处理**：清洗、转换、特征工程
4. **模型选择**：根据问题选择合适的算法
5. **模型训练**：使用训练数据学习参数
6. **模型评估**：使用测试数据评估性能
7. **模型调优**：调整超参数优化性能
8. **模型部署**：将训练好的模型应用到实际场景

## 三、深度学习基础

### 3.1 深度学习的定义
**深度学习（Deep Learning）** 是机器学习的一个分支，使用多层神经网络自动学习数据的层次化特征表示。

**关键特征**：
- **深度架构**：包含多个隐藏层的神经网络
- **端到端学习**：直接从原始数据学习目标函数
- **特征学习**：自动提取数据的多层次特征
- **表示学习**：学习数据的有效表示

### 3.2 神经网络基本组成
1. **神经元（Neuron）**：神经网络的基本单元
   - 输入：$x_1, x_2, ..., x_n$
   - 权重：$w_1, w_2, ..., w_n$
   - 偏置：$b$
   - 激活函数：$f$
   - 输出：$y = f(\sum_{i=1}^n w_i x_i + b)$

2. **激活函数（Activation Function）**：引入非线性
   - **Sigmoid**：$σ(x) = \frac{1}{1+e^{-x}}$，输出(0,1)
   - **Tanh**：$\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$，输出(-1,1)
   - **ReLU**：$ReLU(x) = \max(0, x)$，计算简单，缓解梯度消失
   - **Softmax**：$softmax(x_i) = \frac{e^{x_i}}{\sum_j e^{x_j}}$，多分类概率输出

3. **网络层（Layer）**：神经元的集合
   - **输入层**：接收原始数据
   - **隐藏层**：进行特征变换
   - **输出层**：产生最终预测

### 3.3 前向传播与反向传播
**前向传播（Forward Propagation）**：
- 输入数据通过各层传递
- 每层进行线性变换和激活
- 最终得到预测输出
- 公式：$a^{(l)} = f(W^{(l)}a^{(l-1)} + b^{(l)})$

**反向传播（Backpropagation）**：
- 计算损失函数对参数的梯度
- 从输出层向输入层反向传播误差
- 更新权重以减少损失
- 核心：链式法则

## 四、模型训练与优化

### 4.1 训练过程
**关键概念**：
1. **批量（Batch）**：一次迭代中使用的样本数量
   - **批量大小（Batch Size）**：影响训练稳定性和内存使用
   - 小批量：计算效率与梯度噪声的平衡

2. **迭代（Iteration）**：一个批量的前向传播和反向传播
3. **周期（Epoch）**：完整遍历一次训练数据集

### 4.2 优化算法详解
**梯度下降家族**：
1. **批量梯度下降（Batch GD）**
   - 使用全部训练数据计算梯度
   - 优点：梯度方向准确
   - 缺点：计算量大，内存要求高

2. **随机梯度下降（SGD）**
   - 每次使用一个样本计算梯度
   - 优点：计算快，可在线学习
   - 缺点：梯度噪声大，收敛不稳定

3. **小批量梯度下降（Mini-batch GD）**
   - 折中方案：使用小批量样本
   - 常用批量大小：32, 64, 128, 256

**自适应优化算法**：
1. **动量法（Momentum）**
   - 引入动量项积累历史梯度
   - 公式：$v_t = βv_{t-1} + (1-β)∇θ_t$
   - 效果：加速收敛，减少振荡

2. **RMSprop**
   - 自适应调整学习率
   - 对频繁更新的参数使用较小学习率
   - 对不频繁更新的参数使用较大学习率

3. **Adam（Adaptive Moment Estimation）**
   - 结合动量和RMSprop思想
   - 自适应学习率和动量
   - 实践中最常用的优化器

### 4.3 学习率调度
**学习率（Learning Rate）**：控制参数更新步长
- **太大**：可能越过最优解，甚至发散
- **太小**：收敛缓慢，可能陷入局部最优

**调度策略**：
1. **固定学习率**：整个训练过程保持不变
2. **步长衰减**：每隔一定周期减小学习率
3. **指数衰减**：$lr_t = lr_0 × γ^t$
4. **余弦退火**：学习率按余弦函数变化
5. **热重启**：周期性重置学习率，跳出局部最优

## 五、模型评估与选择

### 5.1 评估指标
**分类任务指标**：
1. **准确率（Accuracy）**：正确预测的比例
   - $Accuracy = \frac{TP+TN}{TP+TN+FP+FN}$
   - 适用：类别平衡的数据集

2. **精确率（Precision）**：预测为正例中实际为正例的比例
   - $Precision = \frac{TP}{TP+FP}$
   - 关注：减少误报

3. **召回率（Recall）**：实际为正例中被正确预测的比例
   - $Recall = \frac{TP}{TP+FN}$
   - 关注：减少漏报

4. **F1分数（F1-Score）**：精确率和召回率的调和平均
   - $F1 = \frac{2×Precision×Recall}{Precision+Recall}$
   - 平衡精确率和召回率

**回归任务指标**：
1. **均方误差（MSE）**：预测值与真实值差值的平方平均
   - $MSE = \frac{1}{n}\sum_{i=1}^n (y_i - \hat{y}_i)^2$
   - 特点：对大误差惩罚更重

2. **均方根误差（RMSE）**：MSE的平方根
   - $RMSE = \sqrt{MSE}$
   - 与原数据同量纲，解释性更好

3. **平均绝对误差（MAE）**：预测值与真实值差值的绝对平均
   - $MAE = \frac{1}{n}\sum_{i=1}^n |y_i - \hat{y}_i|$
   - 特点：对异常值不敏感

### 5.2 验证方法
**数据集划分**：
1. **训练集（Training Set）**：用于训练模型参数
2. **验证集（Validation Set）**：用于调参和模型选择
3. **测试集（Test Set）**：用于最终性能评估

**划分比例**：
- 小数据集（<10k）：70%-15%-15%
- 中数据集（10k-100k）：60%-20%-20%
- 大数据集（>100k）：98%-1%-1%

**交叉验证方法**：
1. **K折交叉验证（K-fold CV）**
   - 数据分为K个互斥子集
   - 每次用K-1个子集训练，1个子集测试
   - 重复K次，取平均性能
   - 常用K=5或10

2. **留一交叉验证（LOOCV）**
   - K=N的特殊情况（N为样本数）
   - 偏差最小，方差最大
   - 计算成本高，适合小数据集

3. **分层K折交叉验证**
   - 保持各类别比例一致
   - 适用于类别不平衡数据

## 六、过拟合与正则化

### 6.1 过拟合现象
**过拟合（Overfitting）**：模型在训练集表现很好，但在测试集表现差
**表现**：
- 训练误差持续下降，测试误差先降后升
- 模型过度记忆训练数据噪声
- 泛化能力差

**原因分析**：
1. **模型复杂度过高**：参数太多，拟合能力过强
2. **训练数据不足**：无法覆盖真实数据分布
3. **训练时间过长**：过度优化训练集
4. **数据噪声大**：模型学习噪声而非规律

### 6.2 欠拟合现象
**欠拟合（Underfitting）**：模型在训练集和测试集表现都差
**表现**：
- 训练误差和测试误差都高
- 模型未能学习数据中的规律
- 拟合能力不足

**原因分析**：
1. **模型复杂度过低**：参数太少，表达能力不足
2. **特征选择不当**：未包含重要特征
3. **训练不充分**：迭代次数不足
4. **学习率不当**：过大导致震荡，过小导致收敛慢

### 6.3 正则化技术
**正则化（Regularization）**：约束模型复杂度，防止过拟合

**L1正则化（Lasso）**：
- 损失函数增加参数绝对值之和
- $L_{new} = L_{original} + λ\sum_i |w_i|$
- 效果：产生稀疏解，自动特征选择

**L2正则化（Ridge）**：
- 损失函数增加参数平方和
- $L_{new} = L_{original} + λ\sum_i w_i^2$
- 效果：减小参数幅度，平滑模型

**弹性网（Elastic Net）**：
- L1和L2正则化的组合
- $L_{new} = L_{original} + λ_1\sum_i |w_i| + λ_2\sum_i w_i^2$
- 兼具特征选择和参数平滑

**Dropout**：
- 训练时随机丢弃部分神经元
- 效果：强制网络学习冗余表示
- 减少神经元间的复杂共适应

**早停法（Early Stopping）**：
- 监控验证集性能
- 性能不再提升时停止训练
- 防止过度优化训练集

## 七、特征工程基础

### 7.1 特征类型
**数值特征**：
- 连续值：年龄、温度、价格
- 离散值：数量、计数
- 处理方法：标准化、归一化、离散化

**类别特征**：
- 名义变量：性别、颜色、品牌
- 有序变量：等级、评分、大小
- 处理方法：独热编码、标签编码、目标编码

**文本特征**：
- 短文本：搜索词、标题
- 长文本：文章、评论
- 处理方法：词袋模型、TF-IDF、词嵌入

**图像特征**：
- 像素值：原始图像数据
- 手工特征：HOG、SIFT、LBP
- 深度学习特征：CNN自动提取

### 7.2 特征缩放
**目的**：使不同尺度的特征具有可比性

**标准化（Z-score）**：
- $x' = \frac{x - μ}{σ}$
- 均值为0，标准差为1
- 适合：特征近似正态分布

**归一化（Min-Max）**：
- $x' = \frac{x - min}{max - min}$
- 范围缩放到[0,1]
- 适合：有界特征，需要固定范围

**鲁棒缩放（Robust Scaling）**：
- 使用中位数和四分位距
- $x' = \frac{x - median}{IQR}$
- 适合：存在异常值
